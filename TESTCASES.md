## Тест-кейсы UI

### Десктоп

| ID        | Название                           | Предусловия               | Шаги                                                                                             | Ожидаемый результат                          | Фактический результат                    | Статус |
| --------- | ---------------------------------- | ------------------------- | ------------------------------------------------------------------------------------------------ | -------------------------------------------- | ---------------------------------------- | ------ |
| 01 | Фильтр по минимальной цене         | Открыта страница `/list`  | 1. Открыть `/list` <br> 2. Найти поле "Мин. цена" <br> 3. Ввести `100` <br> 4. Применить фильтр  | Отображаются объявления с ценой ≥ 1000       | Соответствует ожидаемому                 | Pass   |
| 02 | Фильтр по максимальной цене        | Открыта страница `/list`  | 1. Открыть `/list` <br> 2. Найти поле "Макс. цена" <br> 3. Ввести `500` <br> 4. Применить фильтр | Отображаются объявления с ценой ≤ 10000      | Отображаются объявления с ценой > 10000  | Fail   |
| 03 | Сортировка по цене (возрастание)   | Открыта страница `/list`  | 1. Открыть `/list` <br> 2. Выбрать сортировку "По цене ↑"                                        | Объявления отсортированы по возрастанию цены | Соответствует ожидаемому                 | Pass   |
| 04 | Сортировка по цене (убывание)      | Открыта страница `/list`  | 1. Открыть `/list` <br> 2. Выбрать сортировку "По цене ↓"                                        | Объявления отсортированы по убыванию цены    | Соответствует ожидаемому                 | Pass   |
| 05 | Фильтр по категории                | Открыта страница `/list`  | 1. Открыть `/list` <br> 2. Выбрать категорию (например, "Электроника")                           | Отображаются объявления выбранной категории  | Соответствует ожидаемому                 | Pass   |
| 06 | Тогл "Только срочные" (включение)  | Открыта страница `/list`  | 1. Открыть `/list` <br> 2. Включить тогл "Только срочные"                                        | Отображаются только срочные объявления       | Отображаются также не срочные объявления | Fail   |
| 07 | Тогл "Только срочные" (выключение) | Тогл включен              | 1. Выключить тогл                                                                                | Отображаются все объявления                  | Соответствует ожидаемому                 | Pass   |
| 08 | Кнопка "Обновить"                  | Открыта страница `/stats` | 1. Открыть `/stats` <br> 2. Нажать "Обновить"                                                    | Данные статистики обновляются                | Соответствует ожидаемому                 | Pass   |
| 09 | Кнопка "Стоп"                      | Таймер запущен            | 1. Нажать "Стоп"                                                                                 | Таймер останавливается                       | Соответствует ожидаемому                 | Pass   |
| 10 | Кнопка "Старт"                     | Таймер остановлен         | 1. Нажать "Старт"                                                                                | Таймер запускается                           | Кнопка не кликабельна                    | Fail   |
